什么是计算机网络

Hexo
2021-01-17
算法, 链表

网络层和数据链路层之间：
IP 数据报的首部中没有地方可以用来指明“下一跳路由器的 IP 地 址”。 ⚫ 当路由器收到待转发的数据报，不是将下一跳路由器的 IP 地址填入 IP 数据报，而是送交下层的网络接口软件。 ⚫ 网络接口软件使用 ARP 负责将下一跳路由器的 IP 地址转换成硬件 地址，并将此硬件地址放在链路层的 MAC 帧的首部，然后根据这 个硬件地址找到下一跳路由器
